```bash
1. Go to railway.app
```

```bash
1. Click "+ Add Service"

```

Railway auto-detects these from your database:
```bash

DB_USER=root  
DB_PASSWORD=<auto-generated>
DB_NAME=railway
JWT_SECRET=your-secret-key
CORS_ORIGIN=https://swipesports.co
```

```bash
# Your API will be live at: https://your-app.up.railway.app
```

